# Azure Agentic AI Lab


## Overview


This project demonstrates autonomous AI agents running on Microsoft Azure Virtual Machines using Python.


The lab includes:


- Incident triage agents

- CloudOps automation agents

- Stateful memory agents

- Tool-using AI agents

- Autonomous escalation workflows


## Technologies


- Microsoft Azure

- Azure Bastion

- Python 3

- Linux

- GitHub

- Agentic AI Concepts


## Features


### Autonomous Incident Triage

- Severity classification

- Missing information detection

- Escalation generation

- Safe troubleshooting recommendations


### CloudOps AI Agent

- CPU validation

- Docker checks

- Azure VM operational checks

- Incident simulation


### Memory Agent

- Stateful workflow tracking

- Incident history retention
