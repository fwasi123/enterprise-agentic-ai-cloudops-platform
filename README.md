# Enterprise Agentic AI CloudOps Platform

## Overview

Enterprise-grade Agentic AI platform built on Microsoft Azure Virtual Machines using Python-based autonomous agents.

This project demonstrates AI-driven operational automation, incident triage, memory-enabled workflows, and intelligent cloud troubleshooting for enterprise environments.

---

## Key Capabilities

- Autonomous CloudOps agents
- AI-driven incident response
- Stateful memory workflows
- Intelligent infrastructure diagnostics
- Tool-enabled operational agents
- Cloud health monitoring automation
- Azure VM operational workflows
- Enterprise AI orchestration

## Features

### AI Triage Agent
- Detects operational anomalies
- Simulates incident classification
- Performs intelligent response actions

### CloudOps Agent
- Performs infrastructure health checks
- Validates CPU and memory utilization
- Simulates operational diagnostics

### Memory Agent
- Tracks workflow state
- Maintains operational context
- Simulates persistent AI memory handling

### Autonomous Agent
- Executes AI-driven operational workflows
- Simulates infrastructure automation tasks

### Tool Agent
- Demonstrates AI tool usage patterns
- Simulates enterprise operational commands
